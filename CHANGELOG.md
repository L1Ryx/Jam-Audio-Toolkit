# Changelog

## 1.0.0 - Initial Release

- Added reusable Sound Event assets for SFX, UI sounds, ambience, and loops.
- Added reusable Music Event assets with fades, crossfades, pause, resume, stop, and scene persistence options.
- Added no-code Jam Audio Player and Jam Music Player components for common Unity callbacks and UnityEvents.
- Added the `JamAudio` one-line programmer API for sound and music playback.
- Added random clip selection, pitch/volume variation, and recent-clip repeat avoidance.
- Added human-readable low-pass and high-pass filter controls.
- Added 2D and 3D positioning options for Sound Events.
- Added optional Audio Mixer Group routing.
- Added custom inspectors, validation warnings, setup helpers, and menu items.
- Added the Quick Demo sample scene with demo music and SFX.
- Added offline documentation, third-party notices, and 0BSD license.

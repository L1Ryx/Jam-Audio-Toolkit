# Jam Audio Toolkit Quick Demo

Open `Scenes/Quick Demo.unity`, press Play, then use the Unity UI buttons.

This sample shows:

- A random Sound Event using three SFX clips.
- Three Music Events with fade/crossfade behavior.
- Pause, resume, and stop music controls.
- No-code Unity Button `OnClick` wiring through Jam Audio Player components.

The scene includes one active `AudioListener` on the Main Camera. Unity needs
one active listener in a scene before audio can be heard.
